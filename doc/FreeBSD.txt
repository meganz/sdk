In order to build SDK on FreeBSD, install the following packages:

devel/autotools
security/cryptopp
databases/sqlite3
c-ares
ftp/curl (CARES OPENSSL)
graphics/freeimage
devel/readline (TERMCAP)

This is corresponding configuration for SDK:

./configure --enable-debug --with-cryptopp=/usr/local --with-sqlite=/usr/local --with-curl=/usr/local --with-freeimage=/usr/local --enable-examples --with-readline=/usr/local --with-cares=/usr/local
